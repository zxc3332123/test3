# 如何部署到 Netlify

本專案是一個純靜態網頁 (Static Website)，非常適合使用 **Netlify** 進行免費且快速的部署。

## 準備工作
1. 確認您的資料夾 `20260118_03` 內包含以下檔案：
   - `index.html`
   - `style.css`
   - `script.js`
   - (非必要) `B3L9學習單-1.pdf` (這是原始素材，不影響網頁運作)

## 步驟教學

### 步驟 1：註冊/登入 Netlify
1. 開啟瀏覽器，前往 [Netlify 官網](https://www.netlify.com/)。
2. 點擊右上角 **Sign up** (如果已有帳號則點 Log in)。
3. 可以使用 GitHub, GitLab, Bitbucket 或 Email 進行註冊。

### 步驟 2：進入部署頁面
1. 登入後，您會看到 Netlify 的 Dashboard (控制台)。
2. 在左側選單中點選 **Sites**。
3. 您會看到一個區域寫著：「**Drag and drop your site output folder here**」。

### 步驟 3：拖曳上傳
1. 打開您的電腦檔案總管，找到本專案的資料夾 `20260118_03`。
2. **直接將整個資料夾** 拖曳到 Netlify 網頁上的虛線區域中。

### 步驟 4：等待部署完成
1. 拖曳後，Netlify 會立刻開始上傳並部署。
2. 幾秒鐘後，您會看到類似 **"Site has been deployed"** 的訊息。
3. 畫面上會提供一個綠色的網址 (例如 `https://dazzling-baba-12345.netlify.app`)。

### 步驟 5：自定義網址 (可選)
1. 點擊 **Site settings** -> **Change site name**。
2. 您可以將其修改為更容易記住的名字，例如 `genshin-card-game-demo` (如果沒被使用過的話)。
3. 儲存後，您的新網址就是 `https://genshin-card-game-demo.netlify.app`。
